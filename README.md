# 🌟 Overview

Developers often struggle with **what to build next**.

DevForge solves this by analyzing your GitHub activity and generating **personalized project recommendations** that help improve your real-world development skills.

The platform generates:

• Project ideas
• System architecture diagrams
• Difficulty analysis
• Step-by-step development roadmaps

---

# ✨ Core Features

### 🔍 GitHub Skill Analyzer

Analyze repositories and detect developer skills.

### 💡 Project Recommendation Engine

Suggest portfolio projects based on detected technologies.

### 🏗 Architecture Visualizer

Interactive architecture diagrams for every generated project.

### 🗺 Roadmap Generator

Step-by-step development guide to build the project.

### 📊 Difficulty Analyzer

Classify projects into:

* Beginner
* Intermediate
* Advanced

# 🚀 Getting Started

## Clone Repository

```
git clone https://github.com/YOUR_USERNAME/devforge.git
cd devforge
```

---

# ⚙️ Development Setup

### Backend

```
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Frontend

```
cd frontend
npm install
npm run dev
```
